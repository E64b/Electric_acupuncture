При подаче питания:

1) Настройки в Start Menu:
1.1. При повороте энкодера - выбираем программу.
1.2. При повороте с нажатием - регулируем максимальный ток для всех электродов от 0мА до 500мА.
1.3. Короткое нажатие - запуск выбранной программы. 
1.4. Длительное нажатие - переход в меню настроек программы (если доступно).

2) Настройки конкретной программы:
2.1. При повороте энкодера - изменяем время экспозиции на один электрод.
2.2. При повороте с нажатием - переходим к другом электроду.
2.3. При коротком нажатии - запуск выбранной программы.
2.4. При длительном нажатии - сохранение в долговременную память настроек программы.
